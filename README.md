# Excalidraw Self-Hosted

A self-hosted Excalidraw instance with authentication and board management.

## Tech Stack
- **Frontend:** React + TypeScript + Vite
- **Backend:** Rust + Axum + SQLx
- **Database:** SQLite
- **Deployment:** Docker Compose
